# MediShop - Online Medicine Delivery App

A React Native mobile app for browsing, searching, and ordering medicines online.

## Features
- Browse medicines by category
- Search & filter
- Add to cart
- User authentication
- Order tracking

## Tech Stack
- React Native (Expo / CLI)
- Node.js (backend - if any)
- Firebase / Supabase (optional)

## Installation
```bash
git clone https://github.com/yourusername/medishop-app.git
cd medishop-app
npm install
npx expo start   # or npm start
